This project aims to understand why customers leave the company (churn) and provide insights to improve retention strategies.
The analysis was performed in Python for data cleaning and EDA, and insights were visualized using Power BI.
 
Tools & Technologies

Python (Pandas, Matplotlib, Seaborn, NumPy)

Power BI (Data modeling, DAX, and visualization)

Dataset: Kaggle Telecom Churn Dataset

Steps Involved

Data Cleaning (Python)::

Removed duplicates and handled missing values

Converted columns to correct data types

Exploratory Data Analysis (EDA)

Analyzed churn patterns across gender, tenure, contract type, and monthly charges

Visualized churn distribution and correlation with key features

Data Visualization (Power BI)::

Created interactive dashboard with:

Total Customers

Churn Rate by Payment Method

Churn Rate by Contract Type

Churn Rate by Tenure Group

Churn Rate by Gender and Senior Citizen

Churn Rate by Monthly Charges

Insights:

Customers with month-to-month contracts have higher churn rates.

Senior citizens and those with higher monthly charges churn more often.

Customers on electronic check payment method churn more.

Long-tenure customers are more loyal.
